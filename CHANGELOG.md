## Conventions

### Types de changements
- **Added** : Nouvelles cartes, fonctionnalités, documentation
- **Changed** : Modifications de données ou structures existantes
- **Fixed** : Corrections de bugs ou erreurs
- **Deprecated** : Champs/fonctionnalités à arrêter
- **Removed** : Suppression de fonctionnalités/données
- **Security** : Correctifs de sécurité

### Format des commits
```
[type](scope): description courte

Description détaillée (optionnel)

Cartes affectées:
- Nom: Référence

Références:
- Issue #123
- Fix #456
```

### Versions
Format : `MAJOR.MINOR.PATCH-PRE.BUILD`
- `0.x.x` : En développement
- `1.0.0` : Release stable (200+ cartes, site web, API)
- `1.1.0+` : Features et bugfixes
